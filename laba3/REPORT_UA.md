## Результати роботи

### Реалізовані функції:

1. **Підрахунок кліків:**
   - Кожен клік по кольору збільшує відповідний лічильник
   - Стан зберігається окремо для кожного світлофора
   - Відображення статистики в реальному часі
   - Незалежні лічильники для вертикального та горизонтального світлофорів

2. **Рефакторинг стилів:**
   - Всі inline styles повністю видалено
   - Створено окремі CSS файли для кожного компонента
   - Використано CSS класи та CSS змінні
   - Покращено структуру та читабельність коду

3. **Обробка подій:**
   - Додано обробник onClick для компоненти Light
   - Реалізовано функцію handleLightClick
   - Оновлення стану через setState
   - Використано callback функції для передачі параметрів

4. **Покращення UX:**
   - Додано hover ефект на світла
   - Курсор змінюється при наведенні
   - Плавні анімації переходів
   - Візуальне виділення активних світел

### Технічні деталі:

- **React Hooks:** useState, useEffect
- **Event Handling:** onClick з параметрами
- **CSS Organization:** Модульна структура
- **State Management:** Об'єкт стану для підрахунку
- **PropTypes:** Валідація всіх props, включаючи функції
- **Best Practices:** Відсутність inline styles
